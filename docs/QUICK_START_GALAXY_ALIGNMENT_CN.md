# 银河系对齐快速开始

## 一分钟上手

### 1. 启动应用
```bash
npm run dev
```

### 2. 打开浏览器
访问 http://localhost:3000

### 3. 放大视图
使用鼠标滚轮放大，直到看到蓝色圆圈（约 200,000 - 10,000,000 光年）

### 4. 打开调试器
点击右下角 "🔵 银河系参考系调试" 按钮

### 5. 调整角度
- 拖动滑块调整 X/Y/Z 轴旋转
- 观察蓝色圆圈与银河系图片的对齐情况
- 从多个角度验证

### 6. 保存配置
找到正确角度后，点击 "复制配置" 按钮

## 目标

让蓝色圆圈（外部星团坐标系中的银河系位置）与银河系俯视图和侧视图对齐。

## 关键点

- 🔵 **蓝色圆圈**: 独立的标记，代表外部星团中的银河系
- 🌌 **银河系图片**: 俯视图和侧视图，保持不变
- 🎯 **目标**: 让蓝色圆圈与银河系图片重合
