# 银河系旋转位置调整指南

## 问题描述
银河系外壳相对于太阳系的旋转位置不正确，需要调整银河系的三轴旋转角度。

## 调整工具

### 1. 银河系旋转调试器
我已经创建了一个实时调试工具，可以在运行时动态调整银河系的旋转角度。

**使用方法：**
1. 启动应用程序
2. 缩放相机到足够远的距离（约 1000 光年以上），使银河系可见
3. 点击右下角的 "🔧 银河系调试" 按钮打开调试器
4. 使用三个滑块实时调整旋转角度：
   - **X 轴旋转（俯仰）**: 控制银河系绕 X 轴的旋转（-180° 到 180°）
   - **Y 轴旋转（偏航）**: 控制银河系绕 Y 轴的旋转（-180° 到 180°）
   - **Z 轴旋转（翻滚）**: 控制银河系绕 Z 轴的旋转（-180° 到 180°）

5. 找到合适的角度后，点击 "复制配置" 按钮
6. 将复制的配置粘贴到 `src/lib/config/galaxyConfig.ts` 文件中

### 2. 配置文件位置
银河系旋转配置位于：`src/lib/config/galaxyConfig.ts`

```typescript
export const GALAXY_CONFIG = {
  // ... 其他配置
  
  // 旋转配置（度）
  rotationX: -90,  // 俯仰角度
  rotationY: 30,   // 偏航角度
  rotationZ: 90,   // 翻滚角度
  
  // ... 其他配置
};
```

## 旋转顺序
银河系使用 **YXZ** 旋转顺序（欧拉角），这意味着：
1. 先绕 Y 轴旋转（偏航）
2. 再绕 X 轴旋转（俯仰）
3. 最后绕 Z 轴旋转（翻滚）

## 坐标系说明
- **X 轴**: 指向春分点方向（赤道坐标系）
- **Y 轴**: 指向北天极方向（垂直向上）
- **Z 轴**: 完成右手坐标系

## 调整建议

### 参考点
1. **太阳位置**: 太阳位于银河系中心约 26,000 光年处
2. **银河平面**: 银河平面相对于黄道面倾斜约 60°
3. **银河中心方向**: 银河中心位于人马座方向（赤经约 266°，赤纬约 -29°）

### 调整步骤
1. **粗调**: 使用较大的步长（10-20°）快速找到大致方向
2. **细调**: 使用较小的步长（1-5°）精确调整
3. **验证**: 
   - 检查银河平面是否与太阳系黄道面正确倾斜
   - 确认银河中心方向是否指向人马座
   - 观察银河系旋臂是否自然展开


## 注意事项
1. 调试器仅在开发时使用，不会影响生产构建
2. 旋转角度以度为单位，内部会自动转换为弧度
3. 修改配置文件后需要重新加载页面才能生效
4. 调试器的实时预览不会保存，必须手动复制配置到文件中

## 故障排除

### 银河系不可见
- 确保相机距离足够远（> 1000 光年）
- 检查 `GALAXY_CONFIG.enabled` 是否为 `true`
- 检查透明度配置 `topViewOpacity` 和 `sideViewOpacity`

### 调试器不显示
- 确保已导入 `GalaxyRotationDebugger` 组件
- 检查 `showGalaxyDebugger` 状态是否正确切换
- 查看浏览器控制台是否有错误信息

### 旋转不生效
- 确认 `SceneManager.getGalaxyRenderer()` 返回有效实例
- 检查旋转顺序是否正确设置为 'YXZ'
- 验证角度值是否在合理范围内（-180° 到 180°）
