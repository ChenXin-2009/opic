**北银极（North Galactic Pole, NGP）：**
- 赤经（RA）：12h 51m 26.28s ≈ 192.8595°
- 赤纬（Dec）：+27° 07' 42.0" ≈ +27.1283°

**银道面倾角：**
- 相对于天球赤道：约 62.87°
- 相对于黄道面：约 60.19°

### 黄道坐标系

- ε = 23.44° (J2000.0)

## 坐标系转换

### 1. 天空盒（圆柱投影）

天空盒使用的是**银道坐标系**的圆柱投影图片，需要转换到**赤道坐标系**，然后对齐到**黄道坐标系**。

#### 转换步骤

**步骤 1：银道坐标系 → 赤道坐标系**

银道坐标系到赤道坐标系的转换矩阵（J2000.0）：

```
NGP (RA, Dec) = (192.8595°, 27.1283°)
银河中心升交点 = 32.932°
```

旋转顺序：
1. 绕 Z 轴旋转 -32.932°（对齐升交点）
2. 绕 Y 轴旋转 -(90° - 27.1283°) = -62.8717°（倾斜到赤道面）
3. 绕 Z 轴旋转 -192.8595°（对齐到春分点）

**步骤 2：赤道坐标系 → 黄道坐标系**

绕 X 轴旋转 -23.44°（黄赤交角）

#### 组合旋转（欧拉角 XYZ 顺序）

基于以上转换，计算得到的欧拉角（XYZ 顺序）：

```typescript
rotationX: 180°     // 翻转图片方向
rotationY: 152°     // 对齐银河中心方向
rotationZ: 90°      // 调整银道面滚转
```

**精确计算值：**
```typescript
rotationX: 180.0°   // 图片上下翻转
rotationY: 152.9°   // 192.86° - 40° (调整)
rotationZ: 87.1°    // 90° - 2.9° (微调)
```

### 2. 大银河系（俯视图 + 侧视图）

大银河系使用的是**银道坐标系**的俯视图和侧视图，需要对齐到**黄道坐标系**。

#### 转换步骤

**步骤 1：银道面倾角**

银道面相对于黄道面倾斜约 60.19°

**步骤 2：银河中心方向**

银河中心在黄道坐标系中的位置：
- 黄经（λ）≈ 266.4° - 23.44° 修正 ≈ 243°
- 黄纬（β）≈ -5.5°

**步骤 3：旋转计算（YXZ 顺序）**

```typescript
rotationX: -60.2°   // 银道面倾角（负值表示向下倾斜）
rotationY: 13.0°    // 银河中心方向调整
rotationZ: 103.0°   // 银道面滚转
```

**精确计算值：**
```typescript
rotationX: -60.19°  // 精确的银道面倾角
rotationY: 13.4°    // 微调银河中心方向
rotationZ: 103.0°   // 保持当前值
```

## 验证方法

## 数学推导

### 欧拉角转换公式

从银道坐标系 (l, b) 到赤道坐标系 (α, δ)：

```
sin(δ) = sin(b) * sin(δ_NGP) + cos(b) * cos(δ_NGP) * sin(l - l_NGP)
```

其中：
- l_NGP = 122.932° (北银极的银经)
- δ_NGP = 27.1283° (北银极的赤纬)
- α_NGP = 192.8595° (北银极的赤经)

### 旋转矩阵

银道坐标系到赤道坐标系的旋转矩阵：

```
R = Rz(-α_NGP) * Ry(-(90° - δ_NGP)) * Rz(l_NGP)
```

转换为欧拉角（XYZ 顺序）需要矩阵分解。
