# 项目名称变更公告 / Project Name Change Announcement

## 中文

### 重要通知

我们很高兴地宣布，**CXIC (CXIN Integrated Cosmos)** 正式更名为 **OPIC (Open Integrated Cosmos)**。

### 变更详情

- **原名称**: CXIC — CXIN Integrated Cosmos
- **新名称**: OPIC — Open Integrated Cosmos
- **生效日期**: 2026年4月30日
- **新域名**: https://opic.cxin.tech
- **新仓库**: https://github.com/ChenXin-2009/OPIC

### 更名原因

"Open" 更好地体现了我们的核心理念：

1. **开放源代码** - 采用 Apache 2.0 许可证，欢迎所有人使用、修改和分发
2. **开放数据** - 基于公开的天文数据源（NASA JPL、ESA Gaia 等）
3. **开放协作** - 欢迎社区贡献，包括人类开发者和 AI 助手的协作
4. **开放架构** - 模块化 MOD 系统，支持插件扩展

### 对用户的影响

- **现有用户**: 无需任何操作，所有功能保持不变
- **开发者**: 请更新您的仓库引用和文档链接
- **贡献者**: 新的 PR 请提交到新仓库地址

### 技术变更

| 项目 | 旧值 | 新值 |
|------|------|------|
| 项目名称 | CXIC | OPIC |
| 完整名称 | CXIN Integrated Cosmos | Open Integrated Cosmos |
| 域名 | somap.cxin.tech | opic.cxin.tech |
| GitHub 仓库 | ChenXin-2009/somap | ChenXin-2009/OPIC |
| npm 包名 | cxic | opic |

### 致谢

感谢所有支持 CXIC 的用户和贡献者。我们期待在 OPIC 的新篇章中继续为您提供更好的宇宙可视化体验。

## English

### Important Notice

We are excited to announce that **CXIC (CXIN Integrated Cosmos)** has been officially renamed to **OPIC (Open Integrated Cosmos)**.

### Change Details

- **Previous Name**: CXIC — CXIN Integrated Cosmos
- **New Name**: OPIC — Open Integrated Cosmos
- **Effective Date**: April 30, 2026
- **New Domain**: https://opic.cxin.tech
- **New Repository**: https://github.com/ChenXin-2009/OPIC

### Reasons for Renaming

"Open" better reflects our core philosophy:

1. **Open Source** - Licensed under Apache 2.0, welcoming everyone to use, modify, and distribute
2. **Open Data** - Built on publicly available astronomical data sources (NASA JPL, ESA Gaia, etc.)
3. **Open Collaboration** - Welcoming community contributions, including collaboration between human developers and AI assistants
4. **Open Architecture** - Modular MOD system supporting plugin extensions

### Impact on Users

- **Existing Users**: No action required, all features remain unchanged
- **Developers**: Please update your repository references and documentation links
- **Contributors**: New PRs should be submitted to the new repository address

### Technical Changes

| Item | Old Value | New Value |
|------|-----------|-----------|
| Project Name | CXIC | OPIC |
| Full Name | CXIN Integrated Cosmos | Open Integrated Cosmos |
| Domain | somap.cxin.tech | opic.cxin.tech |
| GitHub Repository | ChenXin-2009/somap | ChenXin-2009/OPIC |
| npm Package | cxic | opic |

### Acknowledgments

Thank you to all users and contributors who have supported CXIC. We look forward to continuing to provide you with an even better universe visualization experience in this new chapter as OPIC.
