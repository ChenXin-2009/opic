# 银河系旋转效果说明

## 为什么 Y 轴和 Z 轴旋转看起来一样？

### 原因分析

银河系的视觉模型由以下部分组成：

1. **俯视图平面（Top View Planes）**
   - 形状：正方形平面（planeSize × planeSize）
   - 数量：3 层（默认配置）
   - 纹理：`MilkyWayTop_Gaia_2100.jpg`（2100×2100 像素）
   - 特点：圆形对称的银河系俯视图

2. **侧视图平面（Side View Planes）**
   - 形状：矩形平面（宽 × 高，比例 5000:1593）
   - 数量：30 个，均匀分布成圆柱形
   - 纹理：`MilkyWaySide_Gaia_5000_2.jpg`
   - 特点：银河系的侧面轮廓

### 对称性问题

由于银河系俯视图是**圆形对称**的：
- 绕 Y 轴旋转（垂直轴）= 在水平面内旋转
- 绕 Z 轴旋转（前后轴）= 在水平面内旋转

**结果：** 对于圆形对称的俯视图，这两种旋转在视觉上是等效的。

### 如何看到明显的旋转差异

#### 方法 1: 使用坐标轴辅助器（推荐）

1. 打开银河系旋转调试器
2. 启用 "显示坐标轴" 开关
3. 观察三色坐标轴：
   - **红色 = X 轴**（指向春分点）
   - **绿色 = Y 轴**（指向北天极）
   - **蓝色 = Z 轴**（完成右手系）
4. 调整旋转时，观察坐标轴的方向变化

#### 方法 2: 观察纹理的非对称特征

银河系纹理虽然大致圆形，但有一些非对称特征：
- 银河旋臂的位置
- 核心区域的亮度分布
- 边缘的翘曲效果

仔细观察这些特征在旋转时的变化。

#### 方法 3: 观察侧视图

侧视图（30 个矩形平面）形成的圆柱形结构：
- 绕 Y 轴旋转：圆柱整体旋转
- 绕 Z 轴旋转：圆柱倾斜

但由于侧视图透明度很低（默认 0.05），可能不太明显。

#### 方法 4: 调整到极端角度

尝试以下组合来看清差异：

**测试 Y 轴：**
```
X = 0°, Y = 90°, Z = 0°
```

**测试 Z 轴：**
```
X = 0°, Y = 0°, Z = 90°
```

**组合测试：**
```
X = 45°, Y = 90°, Z = 0°
X = 45°, Y = 0°, Z = 90°
```

观察银河系相对于太阳系黄道面的倾斜方向。

## 实际应用建议

### 调整策略

由于 Y 和 Z 轴的视觉效果相似，建议：

1. **先调整 X 轴**（俯仰角）
   - 控制银河平面相对于黄道面的倾斜
   - 这是最明显的旋转效果

2. **再调整 Y 轴**（偏航角）
   - 控制银河中心的方向
   - 使银河中心指向人马座方向

3. **最后微调 Z 轴**（翻滚角）
   - 微调银河旋臂的展开方向
   - 配合 Y 轴达到正确的方向

### 天文学参考

**银河系相对于黄道面的方向：**
- 银河平面倾角：约 60° 相对于黄道面
- 银河中心方向：人马座（RA ~266°, Dec ~-29°）
- 北银极方向：后发座（RA ~192.9°, Dec ~27.1°）

**推荐的调整顺序：**
1. 设置 X 轴使银河平面倾斜约 60°
2. 设置 Y 轴使银河中心指向人马座方向
3. 微调 Z 轴使整体方向正确

## 增强可视化

### 提高侧视图可见度

如果想更清楚地看到 Y/Z 轴的差异，可以临时增加侧视图的透明度：

在 `src/lib/config/galaxyConfig.ts` 中：
```typescript
sideViewOpacity: 0.2,  // 从 0.05 增加到 0.2
```

### 增加翘曲效果

翘曲效果会打破圆形对称性：

```typescript
warpEnabled: true,
warpAmplitude: 0.15,  // 从 0.08 增加到 0.15
warpAngle: 45,        // 从 0 改为 45
```

### 添加参考标记

在控制台中添加参考点：

const marker = new THREE.Mesh(

## 技术细节

### 旋转顺序 (YXZ)

Three.js 使用欧拉角，旋转顺序为 YXZ：
1. 先绕 Y 轴旋转（偏航）
2. 再绕 X 轴旋转（俯仰）
3. 最后绕 Z 轴旋转（翻滚）

这意味着后面的旋转是在前面旋转的基础上进行的。

### 局部坐标系 vs 世界坐标系

- 旋转是在**局部坐标系**中进行的
- 每次旋转后，局部坐标轴的方向会改变
- 这就是为什么旋转顺序很重要

### 万向节死锁

当 X 轴旋转接近 ±90° 时，Y 轴和 Z 轴会趋于平行，导致：
- 两个轴的旋转效果重叠
- 失去一个自由度
- 这是欧拉角的固有限制

## 总结

- Y 轴和 Z 轴旋转看起来相似是**正常的**，因为银河系俯视图是圆形对称的
- 使用**坐标轴辅助器**可以清楚地看到旋转差异
- 实际调整时，**X 轴旋转最重要**，Y 和 Z 轴用于微调方向
- 参考天文学数据，银河平面应该倾斜约 60° 相对于黄道面
- 如果需要更明显的视觉差异，可以增加侧视图透明度或翘曲效果
