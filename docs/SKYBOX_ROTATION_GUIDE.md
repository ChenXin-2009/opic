# 天空盒旋转调整指南

## 什么是天空盒？

天空盒（Skybox）是在**太阳系层级**显示的银河系背景图像。它是一个巨大的球体，包裹着整个太阳系场景，使用圆柱投影（equirectangular）的银河系图片作为纹理。

### 与大银河系的区别

项目中有两个银河系视觉元素：

1. **天空盒（Skybox）** - 本指南的主题
   - 显示距离：太阳系层级（< 1000 光年）
   - 用途：提供沉浸式的银河系背景
   - 纹理：`8k_stars_milky_way.jpg`（圆柱投影）
   - 配置位置：`src/lib/3d/SceneManager.ts` 中的 `MILKY_WAY_ORIENTATION`

2. **大银河系（Galaxy Renderer）**
   - 显示距离：远距离视图（> 1000 光年）
   - 用途：显示银河系的整体结构
   - 纹理：`MilkyWayTop_Gaia_2100.jpg` 和 `MilkyWaySide_Gaia_5000_2.jpg`
   - 配置位置：`src/lib/config/galaxyConfig.ts`

## 使用调试器

### 1. 打开调试器

在太阳系视图中（不要缩放太远），点击右下角的 **"🌠 天空盒调试"** 按钮。

### 2. 调整旋转

使用三个滑块调整天空盒的旋转：

- **X 轴（俯仰）**: 控制银河平面的上下倾斜
- **Y 轴（偏航）**: 控制银河系的左右旋转
- **Z 轴（翻滚）**: 控制银河系的滚动角度

### 3. 观察效果

在太阳系视图中观察背景的银河系图像如何旋转。

### 4. 保存配置

找到满意的角度后：
1. 点击 "复制配置" 按钮
2. 打开 `src/lib/3d/SceneManager.ts`
3. 找到 `MILKY_WAY_ORIENTATION` 配置
4. 粘贴新的配置值
5. 保存文件并重新加载页面

## 天文学参考

### 坐标系转换

天空盒使用的图片是**银道坐标系**的圆柱投影，需要转换到**赤道坐标系**：

- **银道面倾角**：约 62.87° 相对于天球赤道
- **银河中心**：RA ≈ 266.4°, Dec ≈ -28.9°（赤道坐标）
- **北银极**：RA ≈ 192.9°, Dec ≈ 27.1°（赤道坐标）

### 调整目标

使天空盒中的银河系方向与实际天文观测一致：
1. 银河平面应该倾斜约 60° 相对于黄道面
2. 银河中心应该指向人马座方向
3. 银河系的视觉方向应该符合从地球观测的效果

## 调整技巧

### 技巧 1: 参考地平线

如果你的场景有地平线或参考平面，可以用它来判断银河平面的倾斜角度。

### 技巧 2: 对比照片

可以参考真实的银河系照片（从地球拍摄）来调整方向。

### 技巧 3: 分步调整

1. **先归零**：点击"归零测试"，观察原始图片的方向
2. **调整 X 轴**：使银河平面倾斜到合适的角度
3. **调整 Y 轴**：使银河中心指向正确的方向
4. **微调 Z 轴**：调整银河系的滚动角度

### 技巧 4: 使用控制台

在浏览器控制台中可以精确设置：

```javascript
// 获取 SceneManager
const sm = window.sceneManager;

// 设置天空盒旋转
sm.setSkyboxRotation(-141.5, 8, 123.4);

// 查看当前旋转
console.log(sm.getSkyboxRotation());
```

## 与大银河系的协调

### 两者应该一致

天空盒和大银河系应该使用相同的方向，这样在缩放时过渡会更自然。

### 调整顺序建议

1. **先调整天空盒**（本指南）
   - 在太阳系层级更容易观察
   - 可以参考行星轨道平面

2. **再调整大银河系**
   - 使用之前调整好的银河系调试器
   - 尽量保持与天空盒一致的方向

### 验证一致性

1. 在太阳系层级观察天空盒
2. 缩放到 1000+ 光年观察大银河系
3. 确认两者的方向大致一致

## 常见问题

### Q: 天空盒看不清楚？

**A:** 天空盒在太阳系层级最明显。如果看不清：
- 确保没有缩放太远（< 1000 光年）
- 调整相机角度，向上或向下看
- 检查天空盒透明度配置

### Q: 调整后没有变化？

**A:** 检查：
1. 控制台是否有日志输出
2. 天空盒是否已加载（纹理加载是异步的）
3. 是否在正确的视图层级

### Q: 旋转效果不明显？

**A:** 因为天空盒是球形对称的，某些旋转可能不明显。尝试：
- 观察银河系的亮带（银河平面）
- 观察银河中心的亮区
- 使用极端角度（0°, 90°, 180°）测试

### Q: 如何恢复默认值？

**A:** 点击调试器中的"重置"按钮，或在代码中恢复：

```typescript
const MILKY_WAY_ORIENTATION = {
  rotationX: -141.5,
  rotationY: 8,
  rotationZ: 123.4,
};
```

天空盒会在远距离时自动淡出。配置在 `SCALE_VIEW_CONFIG` 中：

```typescript

milkyWayBackgroundFadeEnd: LIGHT_YEAR_TO_AU,  // 完全隐藏距离（AU）
```

## 技术细节

### 圆柱投影

天空盒使用的图片是圆柱投影（equirectangular），也称为等距圆柱投影：
- 经度映射到 X 轴（0° 到 360°）
- 纬度映射到 Y 轴（-90° 到 +90°）
- 适合全景图和天球图

### 渲染设置

天空盒使用以下设置：
- `side: THREE.BackSide` - 从内部可见
- `depthWrite: false` - 不写入深度缓冲
- `renderOrder: -1` - 最先渲染（在背景层）

### 性能考虑

- 天空盒是静态的，不影响性能
- 纹理加载是异步的，不阻塞主线程
- 使用 8K 纹理可能需要较多显存
