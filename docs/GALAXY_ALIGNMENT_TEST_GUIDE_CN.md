# 银河系对齐测试指南

## 快速开始

### 1. 启动应用
```bash
npm run dev
```

### 2. 打开浏览器
访问 http://localhost:3000

### 3. 放大到本星系群尺度

使用鼠标滚轮或触摸手势放大，直到看到：
- 蓝色圆圈（银河系平面标记）
- 外部星系（小点）

**目标距离**: 200,000 - 10,000,000 光年

### 4. 打开调试器

点击右下角的 "🔵 银河系参考系调试" 按钮

### 5. 调整角度

观察蓝色圆圈和银河系图片的相对位置：

#### 检查点1: 俯视角度
- 从上方（北银极方向）观察
- 蓝色圆圈应该与银河系俯视图对齐
- 调整 Z 轴旋转偏移

#### 检查点2: 侧视角度
- 从侧面观察
- 蓝色圆圈应该与银河系侧视图对齐
- 调整 X 轴旋转偏移

#### 检查点3: 银河中心方向
- 观察银河中心（人马座方向）
- 调整 Y 轴旋转偏移

### 6. 微调

使用 0.5° 的步进进行精细调整：
- 每次调整后观察效果
- 从多个角度验证
- 记录最佳值

### 7. 保存配置

找到正确角度后：
1. 点击 "复制配置" 按钮
2. 配置会复制到剪贴板
3. 粘贴到文本文件保存

## 预期结果

### 正确对齐的标志

✅ 蓝色圆圈与银河系俯视图重合  
✅ 从侧面看，蓝色圆圈与银河系侧视图平行  
✅ 银河中心方向一致  
✅ 从任意角度观察都保持对齐  

### 常见问题

❌ 蓝色圆圈看不见
- 检查相机距离（应该在 200k-10M 光年）
- 尝试放大或缩小

❌ 蓝色圆圈太暗
- 这是正常的，透明度设置为 0.3
- 可以在 `GalaxyPlaneDebugRenderer.ts` 中调整

❌ 调整后没有变化
- 检查浏览器控制台是否有错误
- 刷新页面重试

## 调试技巧

### 1. 使用坐标轴辅助器

在 `SceneManager.ts` 中临时添加：
```typescript
const axesHelper = new THREE.AxesHelper(1e8);
this.scene.add(axesHelper);
```

### 2. 查看旋转值

打开浏览器控制台，查看实时旋转值：
```
🔧 银河系参考系调试: 更新旋转角度 { x: 0, y: 0, z: 0 }
```

### 3. 对比银河系配置

当前银河系基础旋转：
- X: -60.2°
- Y: 13.4°
- Z: 103.0°

最终旋转 = 基础旋转 + 偏移
