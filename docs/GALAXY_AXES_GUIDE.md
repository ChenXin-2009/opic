# 银河系坐标轴辅助器使用指南

## 功能说明

坐标轴辅助器可以帮助你清楚地看到银河系的旋转方向，即使银河系纹理是圆形对称的。

## 如何使用

### 1. 打开调试器
点击右下角的 "🔧 银河系调试" 按钮

### 2. 启用坐标轴
在调试器面板中，打开 "显示坐标轴" 开关

### 3. 观察坐标轴
你会看到三条彩色的线从银河系中心延伸出来：
- **红色线 = X 轴**（指向春分点方向）
- **绿色线 = Y 轴**（指向北天极方向）
- **蓝色线 = Z 轴**（完成右手坐标系）

### 4. 调整旋转
移动滑块时，观察坐标轴如何旋转：

**X 轴旋转（俯仰）：**
- 绿色（Y）和蓝色（Z）轴在垂直平面内旋转
- 红色（X）轴保持水平方向

**Y 轴旋转（偏航）：**
- 红色（X）和蓝色（Z）轴在水平面内旋转
- 绿色（Y）轴保持垂直方向

**Z 轴旋转（翻滚）：**
- 红色（X）和绿色（Y）轴旋转
- 蓝色（Z）轴保持前后方向

## 理解旋转效果

### 为什么 Y 和 Z 轴看起来相似？

因为银河系俯视图是**圆形对称**的：
- Y 轴旋转 = 在水平面内旋转圆盘
- Z 轴旋转 = 在水平面内旋转圆盘

**但是！** 通过坐标轴，你可以清楚地看到它们的区别：
- Y 轴旋转：绿色轴保持垂直，红色和蓝色轴水平旋转
- Z 轴旋转：蓝色轴保持前后，红色和绿色轴旋转

### 实际意义

虽然视觉效果相似，但在天文学上它们有不同的意义：

**Y 轴旋转：**
- 改变银河中心在天球上的赤经位置
- 影响银河系相对于春分点的方向

**Z 轴旋转：**
- 改变银河系的"滚动"角度
- 影响银河旋臂的展开方向

## 调整建议

### 步骤 1: 设置 X 轴（最重要）
```
目标：使银河平面倾斜约 60° 相对于黄道面
建议值：X = -60° 到 -90°
```

观察：绿色（Y）轴应该倾斜，不是完全垂直

### 步骤 2: 设置 Y 轴
```
目标：使银河中心指向人马座方向（RA ~266°）
建议值：Y = 0° 到 90°
```

观察：红色（X）轴的方向（指向春分点）

### 步骤 3: 微调 Z 轴
```
目标：调整银河旋臂的展开方向
建议值：Z = 0° 到 90°
```

观察：整体方向是否符合天文观测

## 天文学参考

### 坐标系定义

**赤道坐标系（J2000.0）：**
- X 轴：指向春分点（RA = 0°, Dec = 0°）
- Y 轴：指向北天极（Dec = +90°）
- Z 轴：完成右手坐标系（RA = 90°, Dec = 0°）

**银河坐标系：**
- 银河中心：人马座方向（RA ~266°, Dec ~-29°）
- 北银极：后发座方向（RA ~192.9°, Dec ~27.1°）
- 银河平面倾角：约 60° 相对于黄道面

### 转换关系

从赤道坐标系到银河坐标系需要：
1. 绕 Y 轴旋转约 266° - 90° = 176°（或 -4°）
2. 绕 X 轴旋转约 90° - 29° = 61°（银河平面倾角）
3. 绕 Z 轴微调

## 高级技巧

### 技巧 1: 对齐到特定方向

如果你想让红色（X）轴指向特定方向：
1. 先归零所有旋转
2. 只调整 Y 轴
3. 观察红色轴的方向

### 技巧 2: 验证倾角

如果你想验证银河平面的倾角：
1. 观察绿色（Y）轴与垂直方向的夹角
2. 这个角度应该接近 60°（银河平面倾角）

### 技巧 3: 使用控制台精确设置

```javascript
// 获取银河系对象
const galaxy = window.scene?.children.find(obj => obj.name === 'Galaxy');

// 设置精确的旋转（弧度）
galaxy.rotation.x = -60 * Math.PI / 180;  // -60度
galaxy.rotation.y = 176 * Math.PI / 180;  // 176度
galaxy.rotation.z = 0;
galaxy.updateMatrixWorld(true);

// 验证
console.log('旋转（度）:', {
  x: galaxy.rotation.x * 180 / Math.PI,
  y: galaxy.rotation.y * 180 / Math.PI,
  z: galaxy.rotation.z * 180 / Math.PI
});
```

## 故障排除

### 坐标轴不显示
- 确保已启用 "显示坐标轴" 开关
- 确保相机距离足够远（> 1000 光年）
- 尝试缩放相机到更远的距离

### 坐标轴太小或太大
坐标轴长度固定为 50,000 光年。如果需要调整：

```javascript
// 在控制台中手动调整
const galaxy = window.scene?.children.find(obj => obj.name === 'Galaxy');
const axes = galaxy?.children.find(c => c.type === 'AxesHelper');
if (axes) {
  axes.scale.set(2, 2, 2);  // 放大 2 倍
}
```

### 坐标轴颜色不明显
如果坐标轴颜色不够明显，可以在控制台中增强：

```javascript
const galaxy = window.scene?.children.find(obj => obj.name === 'Galaxy');
const axes = galaxy?.children.find(c => c.type === 'AxesHelper');
if (axes && axes.material) {
  axes.material.linewidth = 3;  // 增加线宽（可能不支持）
}
```

## 完成调整后

1. 找到满意的角度
2. 关闭 "显示坐标轴" 开关（可选）
3. 点击 "复制配置" 按钮
4. 粘贴到 `src/lib/config/galaxyConfig.ts`
5. 保存并重新加载页面
