## 🎯 设计理念

### 颜色语义
| 颜色 | 含义 | 使用场景 |
|------|------|----------|
| 蓝色 | 前进、未来、正向 | 时间前进、未来时间 |
| 红色 | 后退、过去、警告 | 时间倒退、错误 |
| 橙色 | 过去 | 过去的时间差 |
| 白色 | 主要内容 | 加载动画、文字 |
| 灰色 | 次要功能 | 按钮、边框 |
| 黄色 | 警告 | 警告信息 |
| 绿色 | 成功 | 成功状态 |

### 为什么这样设计？

1. **直观性** 🎯
   - 蓝色 = 前进（向右）- 符合进度条习惯
   - 红色 = 后退（向左）- 符合警告/停止的直觉

2. **一致性** 🔄
   - 时间滑块和时间差文字使用相同的颜色逻辑
   - 蓝色贯穿所有"前进/未来"的概念

3. **对比度** 📊
   - 黑色背景 + 白色元素 = 最高对比度
   - 蓝色和红色在黑色背景上都很醒目

4. **专业感** 💼
   - 黑白灰为主色调，简洁专业
   - 功能色仅用于语义明确的场景

5. **科学性** 🔬
   - 与太阳的科学白色（5778K）协调
   - 符合天文应用的专业形象
