# 天空盒旋转最终配置

## 调整结果

经过测试和调整，找到了合适的天空盒（太阳系层级银河系背景）旋转角度。

### 最终配置值

```typescript
const MILKY_WAY_ORIENTATION = {
  rotationX: 180,    // 俯仰角度
  rotationY: 152,    // 偏航角度
  rotationZ: 84.5,   // 翻滚角度
};
```

### 配置位置

文件：`src/lib/3d/SceneManager.ts`（约第 34 行）

## 配置对比

### 变化情况

| 参数 | 原始值 | 新值 | 变化 |
|------|--------|------|------|
| rotationX | -141.5° | 180° | +321.5° (或 -38.5°) |
| rotationY | 8° | 152° | +144° |
| rotationZ | 123.4° | 84.5° | -38.9° |

### 主要调整

1. **X 轴（俯仰）**：从 -141.5° 改为 180°
   - 银河平面的倾斜角度完全翻转
   - 使银河系背景的方向更符合观测

2. **Y 轴（偏航）**：从 8° 改为 152°
   - 银河系在水平面内旋转了约 144°
   - 调整银河中心的方向

3. **Z 轴（翻滚）**：从 123.4° 改为 84.5°
   - 减少了约 39° 的滚动角度
   - 微调银河系的整体方向

## 技术细节

### 旋转顺序

天空盒使用 **XYZ** 旋转顺序（与大银河系的 YXZ 不同）：
1. 先绕 X 轴旋转（俯仰）
2. 再绕 Y 轴旋转（偏航）
3. 最后绕 Z 轴旋转（翻滚）

### 坐标系

- **X 轴**：指向春分点（RA = 0°）
- **Y 轴**：指向北天极（Dec = +90°）
- **Z 轴**：完成右手坐标系

### 纹理映射

天空盒使用圆柱投影（equirectangular）：
- 图片的左右边缘对应经度 0° 和 360°
- 图片的上下边缘对应纬度 +90° 和 -90°
- 旋转会改变图片在球面上的映射方向

## 备注

### 为什么配置值差异这么大？

天空盒和大银河系使用：
- 不同的纹理（圆柱投影 vs 俯视图/侧视图）
- 不同的坐标系参考
- 不同的旋转顺序（XYZ vs YXZ）
- 不同的几何结构（球体 vs 平面组合）

因此配置值差异很大是正常的。

### 如何验证两者一致？

1. 在太阳系层级观察天空盒的银河平面方向
2. 缩放到 1000+ 光年观察大银河系的平面方向
3. 两者应该大致平行或一致
